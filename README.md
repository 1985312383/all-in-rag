# All-in-RAG：从零到一的RAG全栈教程

> **「RAG」 检索增强生成技术全栈教程。** 通过体系化的RAG学习路径和动手实践项目，帮助初学者：
> 1. 深入理解RAG技术核心原理和架构设计。
> 2. 掌握RAG全栈技能（从数据处理到生产部署）。
> 3. 构建可落地的智能问答和知识检索应用。

## 项目受众

具备Python编程基础，对RAG感兴趣又无从下手的同学
